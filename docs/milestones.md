Milestones

- Deliverables
  - Architecture design doc, integration points (AIRIS, MAGUS, Atomspace)

  - Preliminary MeTTa output format; sample trace and graph outputs
  - Evaluation plan and benchmarking criteria
- Success

- Success

  - Bidirectional data flow demonstrated; ≥ 95% persistence fidelity
  - At least one test case with improved MAGUS decisions
